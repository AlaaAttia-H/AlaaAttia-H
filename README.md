## Selected Projects

### Embedded Systems & Real-Time Systems
- Developed embedded applications using **AVR microcontrollers**, **ESP32**, **Zephyr RTOS**, and **Raspberry Pi**.
- Worked with Zephyr concepts such as threads, timers, GPIO, Kconfig, device tree overlays, I2C communication, scheduling, and task timing.
- Implemented real-time scheduling concepts including fixed-priority scheduling, acceptance tests, EDF scheduling, and deferrable servers for aperiodic tasks.
- Built a self-balancing ball and beam system using an ATmega microcontroller, PID control, an IR sensor, and a servo motor.

### Robotics & Autonomous Systems
- Designed, controlled, and fabricated a self-driving 6-wheeled rover for hazardous terrain.
- Simulated and implemented autonomous vehicle behavior using **ROS1**, **Gazebo**, Linux, sensors, and trajectory control.
- Modeled and controlled a 5-DOF humanoid service robot arm using MATLAB/Simscape.
- Worked on a playing-cards robot using image processing and an electromagnet end-effector.
- Developed dynamic models for rover systems using bond graph modeling and 20-sim.

### IoT & Hardware–Software Systems
- Built an IoT-based health-tracking smartwatch using GPS, pulse oximeter, OLED display, ESP8266, APIs, cloud triggers, and emergency-call logic.
- Worked on an LVDT-based scanning system for improving prosthetic socket design, including sensor calibration, mechanical design, and 3D model generation.
- Developed a rescue drone concept using Stateflow/Simulink, Raspberry Pi integration, generated C code, and Program-in-the-Loop testing.
- Implemented PLC-based and electro-pneumatic automation systems using industrial sensors, cylinders, and TIA Portal.

### Distributed & Software Systems
- Developed a distributed flight-control-style system where multiple nodes compute control proposals, elect a leader, and apply decisions in total order.
- Used TCP for point-to-point communication, UDP multicast for discovery and heartbeats, multithreading for concurrent node behavior, and the Bully algorithm for leader election.
- Worked on software projects involving Python, Java, OOP, Git/GitHub, Linux, and system-level debugging.
